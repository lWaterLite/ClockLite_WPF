# ClockLite

------

[![ClockLite](https://img.shields.io/badge/ClockLite-%40lWaterLite-blue)](https://github.com/lWaterLite) ![.NetFramework](https://img.shields.io/badge/.NETFramework-v4.7.2-green)

ClockLite 是一个基于WPF的桌面应用，它是一个简单的桌面时钟。

------

在我的个人网站上有一个时钟控件，有朋友希望我能单独将它截出来做成桌面摆件，于是有了这个应用。
